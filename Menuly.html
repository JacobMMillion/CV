Menuly: IOS application

Developed by Jacob Million

I created Menuly in order to attempt to minimize an issue I noticed during my time working in the service industry: unorganization. Most of the time, menu descriptions are stored on something such as a google doc. The goal of Menuly is to allow managers to easily make changes to the menu descriptions, and to allow servers an easy interfance to referance and study. While further implementation is needed in order to fully realize this goal, the general structure of Menuly has been put together.

Further implementations to be made:

-Login functionality, currently can have all login fields blank and log in will be successful
-Seperate manager / staff accounts, staff has view only permissions, while managers can add / remove menu items and subcategories
-File sync, currently menu info is stored locally on each users device. Additional implementation needed to allow for managers to post an update to the menu and give staff ability to sync with this update
